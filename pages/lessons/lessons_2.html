Редактор кода

Цель:

Научиться работать с редактором кода Visual Studio Code (загружать, устанавливать, настраивать).

Задачи:

  • Определить понятие редактора кода
  • Научиться загружать, устанавливать и настраивать редактор
  • Сформировать навык запуска проектов

Материал:

Полезные ссылки:

Visual Studio Code (официальный сайт)

Критерии сдачи:

  • Скачать VS Code
  • Установить расширения: Language Pack, HTML Hint, Live Server
  • Запустить, создать и открыть папку с проектом (из-под VS Code)
  • Создать в папке файлы со следующими названиями и расширениями: index.html, style.css, scripts.js (из-под VS Code)
  • Создать в папке с проектом папки с названиями: css, img, fonts, js (из-под VS Code)
  • Поместить в папку CSS файл style.css, а в папку JS файл scripts.js (из-под VS Code)

Не зачёт:

  • VS Code не загружен и не установлен
  • Отсутствуют расширения Language Pack, HTML Hint, Live Server
  • Не выполнение требований по созданию и наполнению проекта
