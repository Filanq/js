Проект по CSS

Цель:

Закрепление пройденного материала по каскадным таблицам стилей CSS

Задачи:

  • Вёрстка адаптивного макета сайта Woodpecker по заготовке леса и производству пиломатериалов

Материал:

Полезные ссылки:

Figma AOC Codepen (Burger wood) Codepen (Scroll wood) woodpecker

Критерии сдачи:

  • Зарегистрируйтесь в Figma
  • Авторизуйтесь в Figma
  • Скачайте шаблон wood.fig
  • Экспортируйте шаблон в свой аккаунт Figma (Главное меню – Import File)
  • Перейдите в шаблон wood
  • Сверстайте шаблон wood вместе со всеми его адаптивами
  • Для реализации шаблона, скачайте медиа файлы по ссылке
  • Для реализации шаблона, скачайте шрифты по ссылке
  • Для реализации JS функционала бургер меню, воспользуйтесь готовым примером c code.pen
  • Для реализации плавного скрола по якорям, воспользуйтесь готовым примером с code.pen
  • Для добавления интерактива, воспользуйтесь библиотекой AOC

Не зачёт:

  • Выполненная работа по внешнему виду сильно отличается от исходного шаблона
  • Не реализована одна и более секций сайта
  • Не реализован адаптив сайта
  • Реализованный адаптив имеет существенные недочеты
  • Не реализовано бургер меню
  • Не реализован плавный скрол по якорям
  • Не реализован интерактив через использование библиотеки AOC
