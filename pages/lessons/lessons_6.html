Проект по HTML

Цель:

Закрепление знаний по HTML вёрстке

Задачи:

  • Создание многостраничного сайта на тему «Уникальный Я»

Материал:

Полезные ссылки:

htmlbook (Справочник) apsolyamov (Онлайн генератор HTML таблицы) colordrop (Цветовые палитры) squoosh (минификатор изображений)

Критерии сдачи:

Шапка сайта
  • Есть логотип на сайте
  • Логотип является ссылкой на главную страницу сайта
  • Меню одинаковое на всех страницах сайта
  • Меню полноценно (присутствуют все элементы из задания)
  • Все пункты меню рабочие (переходят на свои страницы)
  • «Контакты» являются якорем главной страницы раздела «Контакты»
Раздел «Новости»
  • Присутствует заголовок секции (раздела)
  • Присутствует заголовок, картинка, описание, ссылка «Читать» на страницу конкретной новости
  • Присутствует ссылка «Читать все» на страницу всех новостей
Раздел «Хобби»
  • Присутствует заголовок секции (раздела)
  • Присутствует заголовок, краткое описание и ссылка «Читать» на страницу конкретного хобби
Раздел «Факты»
  • Присутствует заголовок секции (раздела)
  • Присутствует одинаковое лого для всех фактов и их описание
  • Присутствует ссылка «Читать все» на страницу всех фарктов
Раздел «Мечты»
  • Присутствует заголовок секции (раздела)
  • Присутствует список желаний
Раздел «Контакты»
  • Присутствует заголовок секции (раздела)
  • Присутствует подзаголовок «Обо мне» («Я») и описание
  • Присутствует ссылка на страницу Вконтакте
  • Добавлен номер телефона
  • Добавлена дата создания сайта
Общие требования к сайту
  • Чистая семантическая вёрстка (теги применены по назначению)
  • Максимально аккуратный код и приятный дизайн
  • Вёрстка сайта осуществлена только методами HTML5 (без CSS и других ЯП)
  • Макет страницы и разделы выполнены современными тегами и соответствуют стандарту HTML5
  • Корректная структура проекта (папок и вложенных файлов)
  • Изображения, файлы и папки названы правильно
  • Информация разделов представлена в полном объеме и логически конечна
  • Отсутствуют устаревшие теги, если для них есть аналоги HTML5 тегов
  • Есть иконка сайта (видно на вкладках)
  • Есть название сайта (прописан в title)
  • Глобальные ссылки (на сторонние сайты) открываются в новом окне
  • Валидный код (проверить валидность через онлайн сервис https://validator.w3.org)

Не зачёт:

  • Не выполнение хотя бы одного требования из перечня “Критерии сдачи”
