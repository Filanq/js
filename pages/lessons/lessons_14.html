Введение в JavaScript

Цель:

Сформировать понятие о назначении и принципах работы интерпретированного языка общего назначения JavaScript.

Задачи:

  • Определение JavaScripts
  • Отличие JS от Java
  • Области применения
  • Инструменты разработчика
  • Подключение JS и первый проект

Материал:

Полезные ссылки:

learn.javascript (Современный учебник JavaScript) learn.javascript (Введение в JavaScript) learn.javascript (Справочники и спецификации) learn.javascript (Редакторы кода) learn.javascript (Консоль разработчика)

Критерии сдачи:

  • Создайте файл script.js. Внутри напишите строчку кода console.log(‘Привет, мир!’);
  • Запустите сайт PlayCode. Внутри напишите строчку кода console.log(‘Привет, мир!’);
  • Откройте вкладку консоль в DevTools браузера. Внутри напишите строчку кода console.log(‘Привет, мир!’);
  • Узнайте на какой строке допущена ошибка в скрипте страницы bug.html

Не зачёт:

  • Не выполнено хотя бы одно из выше перечисленных заданий
  • Выполненные задания не полноценны
  • Выполненные задания имею существенные недочеты
